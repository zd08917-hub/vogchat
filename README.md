## 📱 О проекте

VOG Chat — это полнофункциональный мессенджер, разработанный на основе современного дизайна из Figma. Приложение предлагает все необходимые функции для комфортного общения: от личных чатов до групповых бесед с поддержкой мультимедиа.

**Оригинальный дизайн в Figma:** [Ссылка на макет](https://www.figma.com/design/Lnfc6uKGg9AOUjGRNFnXk5/%D0%9C%D0%B5%D1%81%D1%81%D0%B5%D0%BD%D0%B4%D0%B6%D0%B5%D1%80-%D0%BD%D0%B0%D0%BF%D0%BE%D0%B4%D0%BE%D0%B1%D0%B8%D0%B5-%D0%A2%D0%B5%D0%BB%D0%B5%D0%B3%D1%80%D0%B0%D0%BC)

## ✨ Функции

### 💬 Основной функционал
- **Личные и групповые чаты** — Общайтесь один на один или создавайте группы
- **Сообщения в реальном времени** — Мгновенная доставка через WebSocket
- **Голосовые сообщения** — Записывайте и отправляйте аудиосообщения
- **Прикрепление файлов** — Отправляйте изображения, документы, видео
- **Поиск контактов** — Быстрый поиск пользователей по имени
- **Создание групп** — Легко создавайте групповые чаты

### 👤 Управление аккаунтом
- **Регистрация по email** — Безопасная аутентификация с 6-значным кодом
- **Профиль пользователя** — Настраиваемый аватар и информация
- **Темная/светлая тема** — Поддержка системных предпочтений
- **Адаптивный дизайн** — Оптимизирован для всех устройств

### 🔧 Дополнительные возможности
- **Редактирование сообщений** — Изменяйте отправленные сообщения
- **Удаление сообщений** — Удаляйте свои сообщения для всех
- **Статусы онлайн** — Видите, кто сейчас активен
- **Уведомления** — Визуальные оповещения о новых сообщениях

## 🏗️ Архитектура

### 📱 Фронтенд
- **React 18** с TypeScript для типобезопасности
- **Vite** для быстрой сборки и разработки
- **Tailwind CSS** для стилизации с утилитарным подходом
- **shadcn/ui** для красивых, доступных компонентов
- **Socket.IO Client** для реального времени
- **React Router** для навигации между страницами

### ⚙️ Бэкенд
- **Node.js + Express** для REST API
- **Socket.IO Server** для WebSocket соединений
- **SQLite** с Knex.js для работы с базой данных
- **Миграции и сиды** для управления схемой БД
- **JWT-токены** для аутентификации
- **Multer** для загрузки файлов

### 🗄️ База данных
- **SQLite** — легковесная, файловая БД
- **Структура**: пользователи, чаты, сообщения, контакты
- **Миграции**: версионное управление схемой
- **Тестовые данные**: готовый набор для разработки

## 🚀 Быстрый старт

### Способ 1: Автоматический запуск (Windows)
```bash
# Запуск всего приложения одной командой
start.bat
```
Батник автоматически:
- Проверит установку Node.js и npm
- Установит все зависимости
- Запустит сервер на порту 3001
- Запустит клиент на порту 5173
- Откроет браузер с приложением

```bash
# Остановка приложения
stop.bat
```

### Способ 2: Ручная установка
```bash
# 1. Клонирование репозитория
git clone https://github.com/zd08917-hub/vogchat.git
cd vogchat

# 2. Установка зависимостей фронтенда
npm install

# 3. Установка зависимостей сервера
cd server
npm install
cd ..

# 4. Запуск сервера (в отдельном терминале)
cd server
npm run dev
# Сервер доступен на http://localhost:3001

# 5. Запуск клиента (в другом терминале)
npm run dev
# Клиент доступен на http://localhost:5173
```

## 🔐 Аутентификация

Система использует безопасную аутентификацию через email:

1. **Регистрация**
   - Введите имя и email
   - Получите 6-значный код (в разработке выводится в консоль)
   - Введите код для подтверждения

2. **Вход**
   - Введите email
   - Получите код подтверждения
   - Введите код для входа

**Важно**: В режиме разработки коды не отправляются на реальный email, а выводятся в консоль сервера для удобства тестирования.

## 🌐 API Endpoints

### Аутентификация
- `POST /api/auth/register` — Регистрация нового пользователя
- `POST /api/auth/login` — Вход по email
- `POST /api/auth/verify` — Подтверждение кода
- `GET /api/auth/me` — Получение информации о текущем пользователе

### Пользователи
- `GET /api/users` — Поиск пользователей
- `GET /api/users/:id` — Получение информации о пользователе
- `PUT /api/users/:id` — Обновление профиля

### Чаты
- `GET /api/chats` — Получение списка чатов
- `POST /api/chats` — Создание нового чата
- `GET /api/chats/:id` — Получение информации о чате
- `GET /api/chats/:id/messages` — Получение сообщений чата

### Сообщения
- `POST /api/messages` — Отправка сообщения
- `PUT /api/messages/:id` — Редактирование сообщения
- `DELETE /api/messages/:id` — Удаление сообщения

## 🚀 Развертывание

### Локальная разработка
1. Убедитесь, что установлен Node.js (версия 18 или выше)
2. Клонируйте репозиторий
3. Установите зависимости (см. [Быстрый старт](#-быстрый-старт))
4. Запустите сервер и клиент

### Production сборка
```bash
# Сборка фронтенда
npm run build

# Сборка сервера
cd server
npm run build
cd ..

# Запуск production сервера
cd server
npm start
```
